# EBA Q&A 2024_7282

## Question

How to report data field B_04.01.0040 if the financial entity is not a branch?

## Background

According to the ITS, the fields B_04.01.0040 is mandatory only if the financial entity making use of the ICT service(s) is a branch of a financial entity (B_04.01.0030). According to the data model, however, this field is marked as a primary key and therefore cannot be missing or reported empty. It is unclear how to fill in the data field without triggering a data quality error when the financial entity reporting is not a branch

## Answer

According to the data model for the Register of Information , data field B_04.01.0040 is marked as a primary key value. As result it should not be missing or reported empty. The field should be filled in with ‘Not Applicable’ if the financial entity is not a branch (i.e., option 2 is selected in B_04.01.0030)
