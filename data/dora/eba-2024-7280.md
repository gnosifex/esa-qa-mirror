# EBA Q&A 2024_7280

## Question

How to report field B_02.02.0150 where the ICT service is not related to storage of data (B_02.02.0140 = 'No')?

## Background

Field B_02.02.0150 is described in the ITS as mandatory if 'Yes' is reported in B_02.02.0140 (Is the ICT service related to (or does it foresee) storage of data (even temporarily)?). According to the data model, however, this field is marked as a primary key and therefore cannot be missing or reported empty. It is unclear how to fill in the data field without triggering a data quality error.

## Answer

According to the data model for the Register of Information , data field B_02.02.0150 is marked as a primary key value. As result it should not be missing or reported empty. The option ‘Not Applicable’ was added to the closed set of options. If the ICT service is not related to storage of data (B_02.02.0140 = 'No'), then the option ‘Not Applicable’ should be used. The field should be filled in with the country of storage of the data otherwise.
