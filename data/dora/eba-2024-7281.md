# EBA Q&A 2024_7281

## Question

How to report data field B_02.02.0160 where the ICT service is not based or does not foresee data processing?

## Background

The ITS describes field B_02.02.0160 as mandatory only if the ICT service is based on or foresees data processing. According to the data model, however, this field is marked as a primary key and therefore cannot be missing or reported empty. It is unclear how to fill in the data field without triggering a data quality error.

## Answer

According to the data model for the Register of Information , data field B_02.02.0160 is marked as a primary key value. As result it should not be missing or reported empty. The option ‘Not Applicable’ was added to the closed set of options. If the ICT service is not based / does not foresee data processing, then the option ‘Not Applicable’ should be used. The field should be filled in with the country of data processing otherwise.
