# EBA Q&A 2014_1025

## Question

On Template C_16.00.b header row 'Subject to ASA' has been numbered on XBRL template as row 125 and inserted above row 130. Header row 'Subject to ASA' should appear on XBRL template C_16.00a above row 110

## Background

Review of Templates

## Answer

While template C 16.00 in Annex I of Regulation (EU) No 680/2014 13 ITS on Supervisory Reporting of institutions (ITS) is correct, the DPM needs to be amended. In particular, "Annotated DPM Table Layout and Data Pont Categorization v2 0 0 with z axes unrolled 13 change highlighted.xlsx" includes a typo in tables C 16.00a and C 16.00b. As on Template C 16.00b header row 'Subject to ASA' has inserted incorrectly above row 130, while it is required in template C 16.00a above row 110 to distinguish banking activities subject to standardized (TSA) approach from banking activities subject to alternative standardized (ASA) approach.
