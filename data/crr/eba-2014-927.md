# EBA Q&A 2014_927

## Question

Rules Annex XV Rules v0559_m Annex XV The annex lists T1 = C 16.00 a & T2 = C 02.00 What the taxonomy validation is doing Actual validation is taxonomy is referring only to C 02..00 and then checking if same value is same. Issue Validation is not working as it is checking same number

## Background

Client is testing all 2000 Validations and are finding differences between Annex XV and the taxonomy

## Answer

In the validation rule v0559_m of Annex XV of Regulation (EU) No 680/2014 13 ITS on Supervisory Reporting of institutions (ITS), the reference to template C 16.00.a is missing in the formula.

V0559_m will be amended to read:

{C 16.00.a, r020,c071} = {C 02.00, r610,c010}

In the same way, v0557_m will be amended to read:

{C 16.00.a, r010,c071} = {C 02.00, r600,c010}
