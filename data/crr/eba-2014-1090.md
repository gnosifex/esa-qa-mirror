# EBA Q&A 2014_1090

## Question

Why "Public sector entities" dimension in the table C 07.00 does not have same type of linking to the table C 02.00 as does have for example "Regional governments or local authorities" dimension?

## Background

There is ID 89540 in C 07.00 and ID 78636 in C 02.00. See "Public sector entities". There is ID 77913 both in C 07.00 and in C 02.00. See "Regional governments or local authorities". In C 02.00 there is no "Counterparty sector" dimension with "Public sector entities" as it has with "Regional governments or local authorities" See Annotated DPM Table Layout and Data Point Categorisation v2.0.0 with z axes unrolled.xlsx

## Answer

All data points referring to the risk weighted exposure amounts after SME factor for the different SA exposure classes in C 02.00 and C 07.00a shall be identically defined in terms of dimensions applied. The DPM will be amended accordingly.
