# EBA Q&A 2025_7435

## Question

For DPM 4.0 the cells {C_08.01.c, r0070, c0130, s*} and {C_08.01.c, r0180, c0130, s*} are identified as identical and have the same VariableID. Considering the instructions for reporting the data in rows 0070 and 0180 different values are expected.

## Background

In document eba_identical_cells_4.0.xlsx the cells {C_08.01.c, r0070, c0130, s*} and {C_08.01.c, r0180, c0130, s*} are identified as identical. According to the instructions for reporting C 08.01, row 0070 Exposure assigned to obligor grades or pools, the exposures for dilution risk of purchased receivables shall not be reported by obligor grades or pools and shall be reported in row 0180 Dilution risk: Total purchased Receivables. This means that values in rows 0070 and 0180 are not expected to be identical.

## Answer

In template C 08.01.c, in the DPM 4.0, the rows 0070 and 0180 have the same modelling, which implies that only one row can be reported. This mistake in the modelling will be amended in the next DPM release for COREP own funds. In the meantime, the data should only be reported for C 08.01.c, row 0070.
